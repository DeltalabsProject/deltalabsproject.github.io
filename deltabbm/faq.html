Faq

Apa perbedaan Clone sama Unclone?

Clone = Tidak akan menimpa BBM Official/yang tersedia di Playstore.

Unclone = Akan menimpa BBM Official/yang tersedia di Playstore.

Mengapa Ikon Notifikasi dan Splat tidak berubah, padahal sudah saya ganti?

Fitur ini hanya berlaku untuk yang menggunakan System UI AOSP. Jadi bagi yang menggunakan system ui selain itu fitur tersebut tidak bisa digunakan. Contoh yang tidak bisa menggunakan di ROM MIUI dan Color OS. Untuk tutorial di MIUI bisa lihat video berikut https://youtu.be/VhboCeUPoEk

Mengapa setiap menutup aplikasi BBM status menjadi Tidak Terhubung/selalu Menyambungkan?

Hal ini dikarenakan pemakaian battery untuk BBM terlalu besar. Biasanya terjadi di ROM MIUI dan Color OS. Untuk mengatasinya masuk pengaturan handphone - Battery & Peforma - Kelola penggunaan battery apl - Matikan Mode Hemat Daya. Atau bisa lihat videonya di https://youtu.be/7VA8JftWMbQ

Mengapa setiap akan mengganti Foto Profil BBM jadi terhenti?

Dikarenakan gallery yang digunakan ga suport, diusahakan menggunakan Gallery pihak 3. Cari saja di Playstore. Biasanya terjadi di ROM MIUI.

Mengapa baru install ketika dibuka langsung BBM Terhenti?

Untuk yang mengalami seperti ini coba dicek system WebViewnya sudah terupdate belum, jika belum silahkan update lebih dulu di Playstore. Namun apabila masih tetap seperti itu silahkan download Delta BBM versi 3.5.5 install, login dan jangan setting apapun juga setelah itu langsung timpa pake Delta BBM versi terakhir.

Mengapa setelah ganti background menggunakan gambar BBM menjadi terhenti?

Disarankan ukuran gambar yang dijadikan background di bawah 1MB jadi sebelum dijadikan background compress dulu file gambarnya.
